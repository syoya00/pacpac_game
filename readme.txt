・操作

w:上に移動
s:下に移動
a:左に移動
d:右に移動
r:リセット（最初からスタート）
ぱくぱくを動かしてアイテムを全て食べましょう。
画面左上からスタートです。
全てのアイテムを食べ終わると次のステージに進みます。
ステージを進めるほどに敵の数が増えていきます。
敵に触れてしまうとミスとなり、そのステージの最初からやり直しです。
